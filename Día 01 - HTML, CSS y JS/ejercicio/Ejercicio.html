Este es el título del artículo

Este es el sector de la pantalla donde va el texto del artículo, nota o post.

El texto será extenso y separado en párrafos. Al pie del artículo se podrán cargar comentarios por los usuarios "registrados". Estos comentarios se mostrarán en orden de carga bajo el artículo.

La sección de comentarios debe cumplir con las siguientes características:

  • Un comentario deberá contener el Usuario que lo cargó, un texto de hasta 200 caracteres y una marca que indique si el comentario se debe destacar.
  • El usuario deberá elegirse desde una lista desplegable.
  • Si un comentario se marcó como "Destacar", en la lista de comentarios se mostrará con un estilo diferente al resto de los comentarios. El estilo es a elección de los desarrolladores.
  • Cómo máximo se podrá cargar 10 comentarios por artículo, la "base de datos" del sitio no tiene capacidad para almacenar más.
  • Un usuario puede cargar más de un comentario, pero no más de 3, para que no monopolizar las opiniones.
  • Un usuario solo puede tener 1 comentario como destacado.
  • Los mensajes de error se mostrarán debajo de los controles de carga del comentario en color rojo, a excepción del máximo de comentarios total, que se mostrará como una popup de alerta.
  • El usuario debe observar el logo de comentarios.
